# gin-admin-cli - [gin-admin](https://github.com/LyricTian/gin-admin)

> GinAdmin辅助工具

## 下载并使用

```bash
$ go get -u github.com/LyricTian/gin-admin-cli
```

### 创建项目

```bash
$ gin-admin-cli new -m -d 项目目录 -p 包名
```

> 示例：
```
gin-admin-cli new -m -d ~/go/src/test-gin-admin -p test-gin-admin
```
